# dhcpcd

1. [TL;DR](#tldr)

## TL;DR

<details>
  <summary>Usage</summary>

```sh
# Start on specific interfaces.
dhcpcd 'eth0'

# Release the current IP address.
dhcpcd -k 'eth0'
dhcpcd --release 'eth0'

# Stop
dhcpcd -x
dhcpcd --exit
```

</details>
