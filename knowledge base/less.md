# Less

## TL;DR

```plaintext
# Search words *forwards* in the current document.
:/keyword ↵

# Search words *backwards* in the current document.
:?keyword ↵

# Toggle case sensitivity in searches.
:-I ↵
```

```sh
# Start with case sensitivity *disabled* in searches
less -I
```
