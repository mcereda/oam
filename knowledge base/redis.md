## License change

Refer to [Redis Adopts Dual Source-Available Licensing].

Starting with Redis 7.4, Redis changed its license from the three-clause Berkeley Software Distribution (BSD) to a dual
_source-available_ licensing model under the Redis Source Available License (RSALv2) and the Server Side Public License
(SSPLv1).
