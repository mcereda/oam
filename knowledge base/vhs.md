## TL;DR

<details>
  <summary>Installation and configuration</summary>

```sh
brew install 'vhs'
docker pull 'ghcr.io/charmbracelet/vhs'
```

</details>

<details>
  <summary>Usage</summary>

```sh
# Check the manual.
vhs manual

# Create new recording files.
vhs new 'demo.tape'

# Process the recording.
vhs 'demo.tape'
```

</details>
